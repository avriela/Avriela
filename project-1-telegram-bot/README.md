# Telegram-бот для отчётов сотрудников

## О проекте
Telegram-бот для автоматизации сбора отчётов 
сотрудников о выполненных работах на объектах.
Интеграция с Google Drive и Google Sheets.

## Моя роль
Функциональное тестирование

## Что тестировала

**Авторизация**
- Вход с ролью сотрудник, администратор, 
  незарегистрированный пользователь
- Корректность отображаемых кнопок для каждой роли

**Пользовательская часть**
- Полный сценарий создания отчёта: 
  выбор объекта → фото → описание → подтверждение → отправка
- Валидации: отправка не фото, превышение лимита 10 фото, 
  отчёт без описания
- Кнопка "Заполнить заново" — сброс данных и возврат к началу
- Кнопки "Назад" на каждом шаге

**Административная часть**
- Добавление и удаление объектов
- Проверка дублей при добавлении объекта с существующим названием
- Добавление и удаление сотрудников
- Проверка дублей по Telegram ID
- Корректность списков объектов и сотрудников
- Поведение при пустых списках

**Логика Excel**
- Первый отчёт за день создаёт новую строку
- Повторный отчёт за день не создаёт новую строку,
  а дописывается в существующую с указанием времени
- Накопительное поле описания — все задачи за день 
  в одной ячейке с временными метками

**Уведомления**
- Напоминания сотрудникам за час до конца рабочего дня
- Сводное уведомление администраторам по окончании дня
  со списком не сдавших отчёт
- Команда /stats — актуальный список сдавших и не сдавших
- Отсутствие уведомлений в воскресенье

**Еженедельный опрос**
- Запуск по расписанию (суббота 16:30)
- Полный сценарий прохождения трёх вопросов
- Подтверждение ответов и отправка
- Кнопка "Заполнить заново" — возврат к первому вопросу
